01 — Architecture
Design Patterns
Four industry-standard patterns implemented throughout the system
◈
Singleton
Ensures only one DataStore instance exists. Private constructor. getInstance() creates lazily. Full serialization support — save to disk, load back.
DataStoregetInstance()save()load()
◇
Factory
Encapsulates user creation in UserFactory.createUser(type, ...). Returns correct subclass (Student, Teacher, Manager, Admin) by type string. Throws IllegalArgumentException for unknown types.
UserFactorycreateUser()StudentTeacher
◉
Observer
News is the Subject — holds List<Observer>. NewsSubscriber implements Observer. When publishNews() is called, notifyObservers() fires all subscribers.
NewsObserverNewsSubscribernotifyObservers()
◫
Strategy
printPapers(Comparator c) and printAllPapers() accept any comparator at runtime. Five interchangeable sorting strategies: by citations, date, pages, GPA, name.
PaperByCitationsPaperByDatePaperByPagesStudentGpa

DataStore · AuthService · Factory
Singleton persistence, authentication, and object creation
◈
DataStore — Singleton
Private constructor + static getInstance(). Implements Serializable. save() writes full object graph to data.ser. load() restores singleton from disk. Holds users, courses, news, requests, messages.
Java
DataStore.getInstance().save("data.ser");
DataStore ds = DataStore.load("data.ser");
// Output: Saved successfully / Loaded successfully
◇
UserFactory — Factory
Static createUser(type, id, username, password, firstName, lastName, gender). Returns Student / Teacher / Manager / Admin. Throws IllegalArgumentException for unknown type.
Java
User u = UserFactory.createUser(
    "student", "S99", "john", "pass",
    "John", "Doe", Gender.MALE
);
((Researcher) u).addResearchPaper(paper);

08 — Error Handling
Custom Exceptions
Domain-specific checked exceptions for business rule violations
InvalidSupervisorException
extends Exception
Thrown when a research supervisor's h-index is less than 3. Only 4th-year students require a supervisor — other years print a message without throwing.
⚡ Student.assignResearchSupervisor() · ResearchUtils.assignSupervisor()
NonResearcherJoinException
extends Exception
Thrown when an object that doesn't implement Researcher tries to join a ResearchProject. joinProject(Object) checks instanceof at runtime.
⚡ ResearchProject.joinProject(Object person)
